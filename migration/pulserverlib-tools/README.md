# pulserverlib-tools

This scaffold contains design/tooling components intended to complement `pulserverlib-interface`.

## Included from monorepo
- `python/pulserver/sequences/`
- `examples/python/`
- `docs/`
- `pyproject.toml`

## Notes
- This package should become an optional dependency of `pulserverlib-interface`.
- Tooling scope can expand to RF, gradient, and sampling-mask helpers.
