# gadgetronlib

This scaffold contains Python runtime components for Gadgetron-oriented server/recon workflows.

## Included from monorepo
- `python/pulserver/recon_server/`
- `docker/`
- `pyproject.toml`, `setup.py`

## Notes
- This package is intended to define/runtime-host recon server contracts.
- Optional high-level recon wrappers can live in a companion `gadgetron-tools` repository.
